## ðŸ“š Materi Golang Level 1

* Variabel & tipe data dasar (`int`, `float64`, `string`, `bool`)
* Operator dasar (`+`, `-`, `*`, `/`, `%`, `>`, `<`)
* Looping (`for`, `for range`)
* Kondisi (`if`, `else`)
* Slice & array (akses index, append)
* Function (parameter & return)
* Logika dasar: sum, max, min, count, rata-rata, frequency
* Reverse slice, ambil angka pertama & terakhir
* Cek apakah angka ada dalam slice
* Hitung jumlah angka genap & ganjil
* Cari index angka tertentu
* Cek slice palindrome
* Hitung jumlah angka lebih kecil dari nilai tertentu
* Buat slice baru dengan angka dikali 2
* Hitung panjang kata dalam slice string
* Ambil elemen string pertama dengan panjang genap
* Filter slice angka lebih besar dari 5
* Hitung jumlah angka dan string di slice campuran (`interface{}`)
* Reverse slice campuran (`interface{}`)
